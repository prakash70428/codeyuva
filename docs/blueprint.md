# **App Name**: CodeYuva Academy

## Core Features:

- Interactive Navbar: Sticky navbar with animated logo, navigation links, and an 'Enroll Now' CTA button that uses smooth scrolling to navigate to the course sections.
- Engaging Hero Section: Hero section with a compelling headline, subheadline, and a 'Get Started' CTA button that navigates users to the Courses section. The visual elements such as animated code snippets will incorporate subtle fade-in and floating effects.
- Course Cards Display: Display course cards with hover effects, each including a syllabus overview, duration, who should join, and an 'Enroll' button.
- Mentors Grid: Mentor cards in a grid layout, each with a photo, name, experience details, and a LinkedIn link, offering a flip card or fade-in effect upon interaction.
- Contact Form: Simple contact form with name, email, and message fields, complemented by a submit button with a loading animation.
- Detailed Footer: Footer containing the logo, a brief description, quick links, contact information, and copyright details for comprehensive site navigation and information.
- FAQ Summarization: An SEO-optimized blog that features articles summarizing popular training questions and challenges faced by those learning DSA.

## Style Guidelines:

- Primary color: Green (#4FE2AC)
- Background color: Black (#080E0C)
- Accent color: White (#F6FCFA)
- Clean, modern, and highly readable typography to ensure information is easily accessible.
- Consistent use of flat, modern icons to represent different sections and features of the platform.
- Responsive layout that adapts smoothly to different screen sizes, ensuring a consistent user experience across devices.
- Subtle and engaging animations throughout the website to guide users and provide feedback.
